Projects

Split L1 + Shared L2 Cache Controller (MESI)

Designed 4-way instruction and 8-way data L1 caches with LRU replacement policy. Implemented inclusive shared L2 cache supporting 4-core architecture. Integrated MESI coherence protocol and validated zero coherence violations through protocol-level verification and performance tracking logic.

MIPS Lite 5-Stage Pipeline Simulator

Developed functional and cycle-accurate timing simulator for MIPS-lite ISA modeling 5-stage in-order pipeline. Implemented forwarding logic to resolve RAW hazards and mitigate branch mispredictions. Performed trace analysis and quantitative speedup evaluation.

Tomasulo’s Algorithm Simulator

Architected dynamic scheduling engine with reservation stations and register renaming. Supported 8+ instruction types enabling out-of-order execution and throughput analysis using latency modeling and cycle-accurate simulation.

Asynchronous FIFO with UVM Verification

Designed 256-depth async FIFO with full, empty, half-full, and quarter-full flags. Mitigated metastability risks and achieved 95%+ functional coverage through reusable UVM-based verification environment.

2x2 Arbiter – Formal Verification

Developed assertion-based verification environment using bind files. Achieved 100% formal closure using VC Formal and JasperGold by identifying and resolving corner-case arbitration bugs.

SPI Master-Slave Controller

Designed FSM-based SPI master and slave modules supporting multi-slave configuration. Verified full-duplex communication and timing compliance using SystemVerilog testbench.
